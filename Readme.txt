Funcionamiento del programa:

Para hacer uso del programa deberá seguir las instrucciones que se muestran por pantalla e ingresar los valores que se le solicitan, los datos que se entreguen deben ser coherentes con los solicitados.
Comenzando por la selección de acción el usuario deberá ingresar un número que corresponda a la acción que quiera llevar a cabo, las opciones actualmente disponibles corresponden a los números 1, 2, 3, 4, 5 y 0.
-Opción 0: Finaliza la ejecución del programa.

-Opción 1: Se le solicita al usuario el nombre de la asignatura que desea ingresar, el usuario deberá entregar una cadena de caracteres (omitiendo el uso de caracteres especiales) la cual pasará a ser el nombre de la asignatura creada.

-Opción 2: El usuario tendrá que ingresar una cadena de caracteres correspondiente al nombre de la asignatura a la que se desea añadir la evaluación, en caso de que se haga de forma correcta y exista dicha asignatura se empezarán a solicitar los siguientes datos de la evaluación:
  Fecha: Corresponde a la fecha en la cual se llevará a cabo la evaluación, esta corresponde a una cadena de caracteres la cual deberá llevar el formato "día/mes/año".
  Unidad: Es el contenido que se evaluará en dicha evaluación, solo puede existir una evaluación por unidad, el usuario deberá ingresar una cadena de caracteres (evitando el uso de caracteres especiales).
  Cantidad de notas: Corresponde al total de notas que se registraron en dicha evaluación, el usuario deberá ingresar un número entero.
  Notas: Se solicitarán todas las notas, según el total solicitado anteriormente, y luego se realizará el calculo correspondiente para finalmente obtener el promedio.
Tras ingresar todos los datos solicitados la evaluación será ingresada en el sistema.  

-Opción 3: El usuario deberá entregar una cadena de caracteres la cual corresponda al nombre de una asignatura, si lo ingresa de forma correcta se mostrarán todas las evaluaciones de dicha asignatura con todos los datos que le corresponden, en caso de ingresarlo de forma incorrecta se le avisará y se devolverá al menú.

-Opción 4: El usuario tras ingresar el número no se le solicitará nada mas y se mostrarán todas las asignaturas y las unidades de todas las evaluaciones que esta posea.
-Opción 5: No se le solicitará nada al usuario y se mostrarán todas las asignaturas registradas en el colegio.
